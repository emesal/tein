Expand description

Stable C ABI function pointer table, populated by the tein host and passed to extensions at init time.

Versioned and append-only — new fields go at the end. Extensions check version before accessing any field.

Function signatures mirror tein::raw::* but use opaque pointer types. The host fills each slot with a trampoline that casts back to sexp and calls the real chibi function.
