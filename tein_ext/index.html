C ABI type definitions for tein extension modules.

This crate defines the stable interface between the tein host and dynamically loaded cdylib extensions. Only C-compatible types cross the boundary — no rust ABI fragility.

Extension crates depend on this crate (and tein-macros), but never on tein itself. The #[tein_module("name", ext = true)] macro generates all the plumbing.
