(tein json) — bidirectional JSON ↔ scheme value conversion.

JSON parsing goes through serde_json::Value (preserving null vs []/{}) then maps to scheme Value. JSON stringifying works directly on raw chibi sexps (bypassing Value::from_raw) to preserve alist structure — chibi’s pair representation collapses (key . val) into a proper list when val is itself a proper list, which would lose the dotted-pair structure needed to detect alists. the 'null symbol distinguishes JSON null from scheme '(). empty [] and empty {} both map to '() — an accepted ambiguity in the alist model; '() stringifies back as [].
