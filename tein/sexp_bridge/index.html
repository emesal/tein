Module sexp_bridge 

Source
Expand description

bidirectional Value ↔ Sexp bridge.

converts between the runtime Value type (chibi-scheme interop) and the pure-rust Sexp AST (serde interop). this is the shared layer that format modules ((tein json), (tein toml), etc.) build upon.
