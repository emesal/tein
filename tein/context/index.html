Scheme evaluation context

Context is a single-threaded Scheme evaluation environment backed by a Chibi-Scheme heap. Create one directly with Context::new() or configure via ContextBuilder for sandboxing, step limits, and environment control.

§Builder pattern

use tein::Context;

let ctx = Context::builder()
    .standard_env()
    .step_limit(100_000)
    .build()?;

§Evaluation

  • Context::evaluate() — evaluate a Scheme expression string
  • Context::call() — call a Scheme procedure with Rust arguments
  • Context::load_file() — load and evaluate a .scm file

§Sandboxing

Four independent layers of control:

  1. Module restriction — ContextBuilder::sandboxed() with [sandbox::Modules]
  2. Step limits — ContextBuilder::step_limit()
  3. File IO policy — ContextBuilder::file_read() / .file_write()
  4. VFS gate — automatic VFS-only when using standard env + sandboxed()

See the sandbox module for module set details.
